⚙️ Prerequisites

Install the following software before running the project.

Node.js

https://nodejs.org

Java 17

https://adoptium.net

Maven

https://maven.apache.org/download.cgi

Git

https://git-scm.com

▶️ Running the Application
Step 1 — Clone Repository
git clone https://github.com/Karthikngk/Dashboard.git
Step 2 — Open Project
cd Dashboard
Step 3 — Start Backend First

⚠️ Always start backend before frontend.

Option 1 — Run Express Backend

Open terminal inside project root folder.

Install dependencies:

npm install

Start backend server:

npm run server

Backend runs on:

http://localhost:8080
Option 2 — Run Spring Boot Backend

Open NEW terminal.

Navigate to Spring Boot folder:

cd backend-springboot

Run Spring Boot application:

mvn spring-boot:run

Spring Boot backend runs on:

http://localhost:8080
Step 4 — Start Frontend Application

Open another NEW terminal.

Go to project root:

cd Dashboard

Run frontend:

npm run dev

Frontend runs on:

http://localhost:3000
🌐 Application URLs
Frontend
http://localhost:3000
Backend
http://localhost:8080
